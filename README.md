# Resell App

This is a mobile application designed to facilitate peer-to-peer buying and selling of used goods.

I am building this platform to create a streamlined marketplace experience, focusing on easy product listing, secure user interactions, and efficient search functionality. It serves as a comprehensive project to demonstrate full-cycle application development.

## Development Workflow

This repository follows a strict branching strategy to ensure stability:

* **master:** This branch contains the stable, production-ready code. It is only updated once features have been fully executed and verified.
* **test:** This is the active development branch. All new functionalities, experiments, and bug fixes are pushed here first.

**Note:** If you are looking for the latest features currently under development, please check out the `test` branch.

## Key Features

* **User Authentication:** Secure login and profile management.
* **Product Listing:** Users can easily upload details and images of items they want to sell.
* **Search & Filter:** Efficient browsing of available products.
* **Marketplace Logic:** Handling the flow between buyers and sellers.

## Tech Stack

* **Language:** Kotlin
* **Platform:** Android
* **Architecture:** MVVM
* **Backend/Database:** Firebase (or your specific backend)

## How to Run It

1.  Clone the repository:
    ```bash
    git clone [https://github.com/Loki-apk/Resell_App.git](https://github.com/Loki-apk/Resell_App.git)
    ```
2.  Switch to the desired branch:
    * For the stable version: `git checkout master`
    * For the latest experimental features: `git checkout test`
3.  Open the project in **Android Studio**.
4.  Sync Gradle and run the application on your device or emulator.
